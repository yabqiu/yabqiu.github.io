本人自工作之日起混迹于中国南方一隅，一个由春天的故事缔造出奇迹的城市里, 一呆 13 载，后来有幸来到大洋彼岸，遂至如今。
以软件编程作为自己唯一的主业，对于技术较为痴迷。可以一问三不知，对触及的知识却不想一知半解，所以凡事都想尽可能的深入了解其原理。

主要的编程语语言是 Java, 近年 AI 兴起，加入了 Python 的大军，最近又迷上了 Rust，其他诸种语言也有所涉猎，希望学能所用。

一直有做笔记的习惯，伴随着 Web 2.0 的兴起，博客便与我不离不弃，成为了我的良师益友。并且我相信对于知识的掌握程度可在经历如下过程层层递进：

阅读学习(10%) -> 听课学习(20%) -> 写作学习(50%) -> 讲课学习(90%) -> 实践、合作来学习(95%)

这就是好记性与烂笔头，因而对学习进行总结、梳理，实践中的问题也记录下来，尽可能的去弄明白，一知半解是很难受的，有机会也会多于大家交流，也包括在这里。

### **关于本站**

年轻那时，是在 QQ 空间里的记录下学习日志，2005 开始在各个博客空间辗转腾挪，其间用过 blogcn.com, 后来搬去了 blogjava.net. 
公共的博客空间纵然是好，因为省却了我们很多无关的事，譬如建立与备份、服务的维护等等，只是缺乏必要的自由度与自我控制的能力.
及至因工作而接触了 WordPress 之后，才开始租用 VPS 主机。关于 VPS 也是不停搬家的故事，从香港，新加坡，马来西亚，到日本的主机都有尝试。

看似很自由，实则太折腾，但不管如何，笔端和思考却一直没有停歇过。本站的博文记录和回溯着对每一个问题的钻研与学习的过程，
写下来也是帮助自己更扎实的理解各方面的知识，也幸与诸位互学共勉。

域名也好，主机也罢，都是要持续付费的，更难长久, 维护也不易。所以近年随着 Github 静态网站 兴起，Hugo 成为我的首选。本站便是用 Hugo 建立的，
维护上省去了很多麻烦, 更能专注于内容的创作。域名上直接用 Github 的 yabqiu.github.io 也无妨，或者绑上 yanbin.blog,
即使 yanbin.blog 不再续费了，yabqiu.github.io 也还存在。

### **关于域名**

最早初立建立时用的域名是 unmi.cc，是在 bloghost.cn 上申请的，后来把域名转出时，却没有任何客服搭理域名提供商的后缀就放弃了。
所以干脆就申请了新的域名 yanbin.blog。因为 unmi.cc 使用有些年头，似乎在网上还有些影响力，因此 unmi.cc 被人抢注之后还被人冒充，
居然有人在用这个域名建立了一个网站，其中的许多日志用了我实实在在的日志标题，但内容全是 AI 生成的，基本没什么价值, AI 时代写博客也太容易了。

但我仍然坚持人工写自己的博客，毕竟这是自己的思考与总结，不是 AI 能代替的。如果能 Vibe blog, 那等于是不要 Blog 了。
