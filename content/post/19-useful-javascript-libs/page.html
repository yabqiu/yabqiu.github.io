流行的 JavaScript 库有jQuery，MooTools，Prototype，Dojo和YUI等，这些 JavaScript 库功能丰富，加上它们众多的插件，几乎能实现任何你需要的功能，然而需要实现一些特定的功能，则可以选择功能更专一的轻量库，今天这篇文章与大家分享16个很有用的 JavaScript 库。

Blackbird: Open Source JavaScript Logging Utility

Blackbird 是一款非常酷的 JavaScript 调试工具，带有一个漂亮的界面显示和过滤调试信息。

Treesaver.js

Treesaver 是一个用于创建杂志布局的 JavaScript 框架。

{{< bundle-image src="2145132645162381.jpg" width="500px" >}} Bibliotype

Bibliotype 是一个简单的基于 HTML、CSS 和 JS ，用于快速原型制作的的框架。

{{< bundle-image src="2145132645162381.jpg" width="500px" >}} Highcharts – Interactive JavaScript charts

Highcharts 是一个纯 JavaScript 编写，用于生产交互式图表的 JavaScript 库。

{{< bundle-image src="2145133311060747.jpg" width="500px" >}} jStat: a JavaScript statistical library

Stat是一个采用JavaScript开发的统计工具包，可以让你执行高级统计操作，而不需要专门的统计语言。

{{< bundle-image src="214513489556039.jpg" width="500px" >}} highlight.js

Highlight.js 是一个用于高亮显示程序代码的 JavaScript 库，简单易用。

{{< bundle-image src="2145135679823174.jpg" width="500px" >}} Respond.js: Fast CSS3 Media Queries for Internet Explorer 6-8 and More

这个快速、轻量的库用于让IE6-8支持 CSS3 Media Queries，是一种让网页支持小屏幕设备显示的特性。

{{< bundle-image src="2145136395918402.jpg" width="500px" >}} Underscore.js

Underscore 是一个实用的JavaScript工具库，提供了类似 Prototype 功能的编程支持，但没有对 JavaScript 内置的对象进行扩展。

{{< bundle-image src="2145137271871342.jpg" width="500px" >}} Popcorn.js

Popcorn.js 是一个HTML5 Video框架，它提供了易于使用的API来同步交互式内容。

{{< bundle-image src="21451481985703339.jpg" width="500px" >}} SlickGrid!

SlickGrid 是一个用于构建简单的、快速、灵活的表格的库。

{{< bundle-image src="21451491719477553.jpg" width="500px" >}} JavaScript InfoVis Toolkit

The JavaScript InfoVis Toolkit 用于创建交互的可视化数据。

{{< bundle-image src="21451410266539015.jpg" width="500px" >}} Handlebars.js: Minimal Templating on Steroids

Handlebars 是一个 JavaScript 页面模板库，帮助你轻松的构建语义化模板。可参考ThinkVitamin的文章

 Getting Started with Handlebars.js。

{{< bundle-image src="21451411275755516.jpg" width="500px" >}} Backbone.js

Backbone.js 是一个 JavaScript MVC 应用框架，通过Models进行key-value绑定及自定义事件处理。

{{< bundle-image src="214514121765847993.jpg" width="500px" >}} OpenFaces

OpenFaces 是一个包含AJAX-powered JSF组件库，一个Ajax框架和一个客户端校验框架的开源框架，它源于QuipuKit并增加了一些新的组件和特性。

{{< bundle-image src="214514131416475081.jpg" width="500px" >}} d3.js

D3 可以让你绑定任何数据到文档对象模型(DOM)，然后再应用数据驱动传递给文档。

{{< bundle-image src="21451414385856415.jpg" width="500px" >}} Visual Event JS Library

Javascript 是事件驱动的语言，这个可视化事件库用于帮助你追踪事件。

{{< bundle-image src="21451415225858013.jpg" width="500px" >}} Tempo: JSON rendering engine

Tempo 节奏是一个小的JSON渲染引擎，让你使用纯HTML就可以构建数据模板。

{{< bundle-image src="214514161989902020.jpg" width="500px" >}} Art Deco – Selectable Text

实现字母分离效果的库，保留了文本能被选择的特性。

{{< bundle-image src="21451517322409600.jpg" width="500px" >}} Using Less.js to Simplify Your CSS3

LESS 是一个扩展CSS，让其拥有动态语言特性的库。
