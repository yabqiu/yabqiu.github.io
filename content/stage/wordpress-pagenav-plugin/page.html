安装的话就是下载下来上传安装，或者直接在 WordPress 中在线安装，然后一般是修改主题对应的 footer.php，在该脚本的最前面或是某处加上一行 PHP 脚本：

<?php if(function_exists(‘wp_pagenavi’)) { wp_pagenavi(); } ?> 或者是：<?php wp_pagenavi(); ?>

当然你可以在任何想显示分页导航条的地方加上面的语句，加在 footer.php 中通常是一种惯例。另外，你可以修改插件里的 pagenavi-css.css 文件来随意定义自己想要的样式。

像其他分页插件还有 pagebar，以及 jQuery 版的 jQuery Pagebar，这个应该也还不错 Zamango Page Navigation。还是看看它们各个长什么样。
pagebar:
jQueryPagebar: 用个滑动条来翻页的，各部分颜色都可定义的，很有意思，但放到我的站点上不那么合适。

虽然它可以自动在页面某处加上该翻页条，可是智能性太差，还是建议在 footer.php 找个地方加上 <?php jquery_pagebar(); ?>。

Zamango Page Navigation 我还没看到界面，当前版本应该是在 WordPress 3.0.1 上不被兼容，激活后提示错误：The plugin generated 38835 characters of unexpected output during activation. If you notice “headers already sent” messages, problems with syndication feeds or other issues, try deactivating or removing this plugin. 并且把一堆的 PHP 源代码给倒了出来。

个人还是建议用 WP-PageNavi，然后需要定制化界面自己调样式文件 pagenavi-css.css 就行。
