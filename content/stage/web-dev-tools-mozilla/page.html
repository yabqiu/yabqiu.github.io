对于很多Web开发人员来说，数目庞大的开发工具往往让人不知如何选择，一些非常实用的工具也因为无法检索而与开发人员失之交臂，Mozilla实
验室近日推出了一种服务帮助解决这一问题，开放Web工具录（Open Web Tools
Directory）可以为Web开发人员提供最新最全面的开发工具。
开放Web工具录网站目前还处在初期阶段因此提供的工具列表并不是十分全面，Mozilla开发工具小组Ben
Galbraith提醒用户，只有拥有Canvas功能的新浏览器才能浏览这些，比如Firefox 3.5、Safari 4、Chrome
2或是Opera 9。

除了提供最新的工具外，该服务还提供了初步的搜索功能，你可以搜索所有文件也可以有选择性的搜索，比如仅搜索代码。为了让其他浏览器用户也能享用到该服务，Mozilla还提供了另一版本的工具录入口。开发人员也可以上传自己推荐的工具，并对该工具进行评级和说明。
