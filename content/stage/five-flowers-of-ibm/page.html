--- title: "IBM软件倾力打造5朵小花 10年完成45项收购 [转]" url: /five-flowers-of-ibm/ date: 2006-01-23T21:52:00-06:00 featured: false draft: true type: post toc: false # menu: main usePageBundles: true categories: - Gossip tags: - IBM comment: true codeMaxLines: 50 # additional wpPostId: 588 wpStatus: publish views: 305 lastmod: 2010-08-28T05:53:24-05:00 ---     当郭士纳(Lou Gerstner)在Comdex大会上用一场划时代的讲演，展望以Web为中心的计算技术的前景后，IBM软件集团应运而生。从1995年成立到今年，IBM软件集团已经走过了10个年头，并且，IBM软件集团已经成为全球第二大软件实体。

　　上世纪90年代中期，是IBM历史上的一道分水岭。以PC为代表的信息产业革命根本改变人们的工作和生活方式，一大批IT企业抓住机遇，崛起于阡陌之间。然而那时的蓝色巨人却似乎已经失去了往日容光，IBM固守着它的硬件，在新的信息产业革命的潮流到来之际，驻足不前。在爆炸般成长的新生市场中，IBM几乎没有取得令人印象深刻的成功。

　　1993年，郭士纳出任IBM董事会主席兼首席执行官，他为IBM带来了压力，也带来了新鲜空气，还有IBM久违的士气。后来的事实证明，郭士纳不仅挽救了IBM，而且使它重新登上信息时代的霸主宝座。1995年，Comdex大会上， Lou Gersmer成为首个描绘电子交易的人。同年，IBM软件与电子商务背宄銎鹋芟撸琁BM软件集团诞生了。如今的IBM软件业务利润率已高达81%，占集团总利润的三分之一。

　　在IBM大中华区副总裁及大中华区软件集团总经理宋家瑜看来，IBM 走路一条不平凡的路，十年间留下了太多值得炫耀和回味的数据，但这些数据毕竟已经成为过去。但至少有五种颜色值得记住，那就是IBM软件五大品牌：紫色的WebSphere、绿色的DB2、黄色的Lotus、红色的Tivoli和蓝色的Rational，如今的五朵小花均在各自领域绽放，传播着IBM软件集团的芬芳。

　　红、黄、蓝、绿、紫，IBM软件在十年间种出了五朵颜色各异的花。值得一提的是，IBM的五朵小花有自己培养的，也有嫁接来的，如今正在高唱整合的IBM软件集团自己当然也深谙整合之道。在过去的10年时间里，IBM软件共完成了45项收购，包括新近收购的DataPower公司。

　　红色的Tivoli、黄色的Lotus和蓝色的Rational同样来自收购，众所周知，红黄蓝融合在一起显现黑色，而黑色正是 "2005 IBM整合实战论坛"的主色调，当整合成为IBM软件的主导业务时，收购的重要性也自然而然凸现出来。不可否认，IBM拥有很多先进的技术和理念，但IBM同样拥有丰富的收购经验。

　　时下，IBM还在继续着对自己有利的收购，而其众多竞争对手也欲通过收购效仿它的成长之路。

　　IBM软件集团成长大事记：

　　1995年，IBM以29亿美元收购了Lotus开发公司。当时，Notes系统的用户总数总计220万，而今天的数字是8500万。

　　1996年，IBM以7.43亿美元收购了Tivoli，当时Tivoli的年度收益为5千万美元，到了1997年，其收益已超过10亿美元。

　　拥抱Java，IBM得到Sun公司的JavaOS和HotJava授权，并建立了业界首家Java认证中心，表明IBM对Java技术和程序的支持，以及对网络计算的承诺。

　　1997年，Tivoli增长达到2100%：Tivoli的收入超过10亿美元。Tivoli收购了Unison Software和Software Artistry两家公司。

　　IBM发布DB2通用数据库第五版，当时是业界第一个完全可升级的Web数据库管理系统。

　　1998年，Tivoli 收购了Mercury Storage Systems 公司，并将其加入存储解决方案部门。

　　IBM 发布集成的软件套件，使Windows NT用户能够构建、部署和管理商业应用程序，同时使IBM成为 Windows NT解决方案的首要供应商。

　　1999年，拥抱 Linux，在 LinuxWorld 大会上，IBM 宣布支持这一急剧成长的开放源码操作系统。规划已久的主要联盟的公告、旗舰产品以及业内最完整的服务，使IBM成为唯一可提供Linux软硬件解决方案及技术支持的公司。

　　2000年，普及 Linux，IBM 加强对 Linux 的支持和开放源代码运动，宣布将使其所有的服务器平台支持 Linux；重申其与 Linux 团体的合作，帮助 Linux 发展并使 IBM 的技术用于 Linux 和开放源代码团体。

　　软件集团的创始人John Thompson 接任新职位，成为董事会的副主席。John 将领导的激情传给了 Steve Mills，后者继任软件集团的高级副总裁和集团执行官。

　　2001年，IBM扩展自己的战略联盟主动权，来为世界范围内各个产业和市场里的独立软件开发商出创造新的商业机会。"Lite 联盟"就是一种全新的商业合作伙伴协议，可以帮助更多不领域的开发商与IBM一道，通过更灵活的形式在更明确的目标地域，共同开拓和应对市场。

　　2002年，IBM 收购 CrossWorlds 软件公司，并将 CrossWorlds 的产品集成到 WebSphere 软件平台。IBM 还收购了Metamerge公司，以进一步增强 IBM 在目录服务器市场的地位。

　　Tivoli收购TrelliSoft和Access3600，Tivoli加强了其安全和存储管理业务。

　　IBM收购了Holosofx，进一步扩大自己的商务集成产品系列。

　　2003年，IBM以21亿美元完成了对Rational Software的收购，使其成为IBM旗下的第五大软件品牌，五朵小花的骨架基本形成。

　　2004年，收购Candle、Trigo、Alphablox和Cyanea。

　　2005年，收购脚步加快，十个月中分别收购SRD、Ascential、Gluecode、Isogon、DWL、DataPower六家公司。
