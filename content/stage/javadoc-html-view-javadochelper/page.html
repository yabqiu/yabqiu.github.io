可是对于大多数的 Java 组件都只提供 HTML 版的 JavaDoc API 帮助，查阅起来很不顺手。没有快速索引查找的功能，而且看不同组件的 API 帮助需要同时打开多个窗口。

下面向大家介绍一款软件，JavaDOCHelper，它不仅能提供像 CHM 文件那般快速索引查找，而且提供一个界面里查阅多个 Java Doc API。该软件未注册版本，只限 30 次的查找功能的使用（不是指索引），现已对该软件完成了破解，并提供更换安装目录时更方便的提示功能，以及免安装，EXE 文件无控制台方式启动。

提供给大家下载，按照 JavaBlog 的要求，切分成了两个包，解压后，直接运行 bin 目录中的 JavaDOCHelper.exe 就行，建议为它在桌面建一个快捷方式，用 Java 的话，组件特别多，时常要查看 API 说明的。

下载地址：  http:///unmi.cc/downloads/JavaDOCHelper1.5.1.rar



可以同时选择多个组件的 JavaDoc Api html 格式的文档，只需要指向 API 文档起努目录，并可设定能按什么方式查找。可设定首页面，即软件一打开所展示的页面。


上图展示了 Index 方式查找，与 CHM 版的帮助文档一样的，只需要输入类的前几个字母就会即时提示；Search 方式，可依据方法名，字段名；包展示，还有其他更多的功能，用了就会有所体验的。

在左上角的 Active API Item 下拉框可以选择搜索或显示那些组件的 API。我觉得它提供最方便的功能就是查阅多个组件的帮助不需要切换页面。

当今的 Java 组件特别多，我在平时使用在把组件及源代码是下载到一个 Component Library 目录，前天一看才知道，竟然是将近4个G了，Java 的组件基本都有一个 HTML 格式的 API 文档，所以相信这个 JavaDOCHelper 对像来说一定派得上用场。

后记：我也是有幸在 2004 年的时候在作者的网站上下载来的这个软件，但是自那不久，那个网站的域名停啦，这个软件在网上也是无处可寻，可能是由于作者的某些原因吧。现在留在手上的可谓是绝版。
