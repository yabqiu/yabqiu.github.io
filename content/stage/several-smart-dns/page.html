例如，对于同一个主机有两个 IP，分别为 192.168.0.1 和  172.16.0.1，域名为 www.unmi.cc, 当然用不同的 IP 访问会走不同的路由线路的。希望电信用户进来访问 www.unmi.cc 是走 192.168.0.1，而网通用户访问 www.unmi.cc 时走 172.16.0.1，默认为 192.168.0.1。

所幸网络上有免费的几个双线智能域名解析系统可使用，下面列出：

1. DNSPod，还记得它曾遭受过暴风影音的殃吧，这也证明它用户多吗。面向国际的是 http://www.dnspod.com，它不支持双线解析。中国用户请用 http://www.dnspod.cn，支持双线解析，并且响应还很快的。它们间的用户是不共享的，可以说是完全独立的，分免费和 VIP 付费用户，看起来似乎很强大。

http://www.dnspod.com 的 dns 是 ns1.dnspod.net  -- ns6.dnspod.net
http://www.dnspod.cn 的 dns 是 f1g1ns1.dnspod.net 和 f1g1ns2.dnspod.net

我用过，可是被非法后清退了。

2. http://www.everdns.cn

EverDNS是和空间服务商第一商务合作的，该服务不计划赢利，也未计划收费。官方承诺能至少能提供一到两年的服务。这个承诺不知道是什么时候作出，从中国 Z)(F 的种种行为来看，承诺基本是放了一个气。光界面上看不是很专业。

刚刚在它的官方网站看到这样的通知：

重要通知: 鉴于大量用户使用everdns的服务从事非法用途并导致大量攻击，EVERDNS即将关闭免费用户的解析以及新用户注册功能，请各位用户尽快转移解析，以免产生不必要的损失！

这也证明了那句话，天下没有免费的午餐，一切都是神马借口。目前仍然可注册并使用，想继续免费使用的请绕道而行。至于合法与非法全赖他们自己裁定，就像是人民还是敌人也是与时俱进的。它的 dns server 是 ns5.erverdns.com 和 ns6.everdns.com。

3. http://www.dnsmax.cn

我还只是听说而已，它的网站我还未能成功访问到，不多加评说，dns server 是 ns1.dnsmax.cn 和 ns2.dnsmax.cn。何时能用时再体验。

看网络上介绍，怎么曾经有一个 www.dnsma.com 也能支持双线解析，只是这个域名已经不干这个活了，已停止服务。

4. http://www.soodns.com

号称将免费 DNS 进行到底，访问它后会重定向到 http://219.235.1.32:8080/ 上去，不知为何不用域名，难道是做鞋的没鞋穿，盖房子的没房住，而且还是使用 8080 端口，未备案不成。还未曾试，界面以及网站上的介绍看还不错，功能也很齐全，协议很标准也很吓人似的，注册都要几十秒钟就行。设置也很简单，分网通、电信和移动三条线路选择，可以去试用下。

它的 dns server 是 ns1.soodns.com 和 ns5.soodns.com

5. WinMyDNS 软件

网络上那东西如果能用国外的应该最好了，可惜双线智能解析太那个中国特色了，只能用国货。如果你不想在别人的平台上，也许你可以自己搭建一个智能的 dns 服务器。当线双线那玩意也只有国人去开发，可以用 WinMyDNS 软件在 Windows 平台下搭建。它的中文站是：http://www.winiis.com，英文站是：http://www.winmydns.com，英文版同样也支持双线解析。

看着它的自我介绍，似乎还能与 DNSPod 媲美，有智能双线解析、自动宕机检测，负载均衡，TXT 文本解析、中文域名等功能，还可二次开发。只是没怎么看出它的授权方式是怎么，是否可免费使用，免费可使用多久等等。有说是免费版，有说共享版，且官网上明显有个付款链接，有兴趣的可以试着安装下，然后把 dns 设置成你的服务器公网的 IP。

已确定，WinMyDNS 是要钱的，免费版可试用一个月，且每改动 DNS 都必须重启 WinMyDNS 服务。它分个人版 1000元，企业版 3000 元，高级企业版 1万，电信专用版 3 万。看这价格说明 WinMyDNS 还是很有实力的。

6. 另外，其他的 DNS 软件，像 Bind 也可以让它支持双线解析，要求你把用户网通的 IP 段尽可能的收集齐全，来者为列表里的网通用户解析为网通 IP，余则认定为电信用户，或者反之。Bind 是个很优秀的软件，可喜的是在 Windows 下平台也有它相应的版本 nebind，下载链接：http://ftp.isc.org/isc/bind8/contrib/，配置方式肯定也是一样的，也是能让它支持双线的。

想要找更多的免费 DNS 软件，请 Google 搜索 linux dns 软件 或 windows dns 软件，选自已最合适的，最好支持双线的。
