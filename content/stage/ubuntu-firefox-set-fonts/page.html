--- title: Ubuntu 下为 Firefox 设置中文字体 url: /ubuntu-firefox-set-fonts/ date: 2011-05-02T01:12:44-05:00 featured: false draft: true toc: false # menu: main usePageBundles: true categories: - Linux/Unix tags: - Linux - FireFox - Ubuntu comment: true codeMaxLines: 50 # additional wpPostId: 3529 wpStatus: publish views: 4106 lastmod: 2013-12-31T02:20:32-06:00 --- 默认风格下 Ubuntu 的 Firefox 浏览中文网页十分难看，字黑黑的不清楚，所以需要设定一下 Firefox 中的字体，幸好选择了中文语言包中系统中有文泉字体可用。网上找到了一篇 Firefox 中文字体的参考，如下：
1.编辑CSS文件，参考：

http://wiki.ubuntu.org.cn/Firefox_%E5%AD%97%E4%BD%93
(声明：本链接内容更适用于，在Firefox首选项中无法设定中文字体的情况！)

2.运行Firefox3.6.13

3.依次选择 Edit \ Preferences \ Content \ Advanced

我的选择:

Fonts for : User Defined

Proportional: Sans Serif      Size:18

Serif: WenQuanYi Bitmap Song

Sans-serif: WenQuanYi Bitmap Song

Monospace: WenQuanYi Bitmap Song      Size:16

Minimum font size:12

Default Character Encoding: Unicode(UTF-8)

Edit \ Preferences \ Content 中的Languages，点Choose

我添加了：English/United States [en-us]

Chinese/China [zh-cn]

重新启动Firefox

操作完之后，可以看到 Firefox 中的中文要好看些了，可是仍然没有 Google Chrome 在 Ubuntu 下浏览中文网页好看。好像还是有剧齿一样，改成与 Google Chrome 一样的字体也无济于事，不知是否还和系统有关
