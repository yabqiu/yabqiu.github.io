[caption id="" align="aligncenter" width="500"] Scala Call-By-Name vs Call-By-Value[/caption]

要指定函数参数 y 以 Call-By-Name 传递，要这么定义 def test(x: Int, y: => Int) = x * x，这样就避免了 test(3, 2*4) 调用时 2*4 被求值，这里把 y 转换成了函数参数 y: => Int。
